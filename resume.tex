\subsection{C++}
\cventry{Spring 2025}{SimOS: Process \& Memory Manager}{Individual Project}{}{}{
An OS simulation through a simplified OS kernel.\newline{}
It simulates process scheduling, memory allocation, and disk I/O handling.\newline{}
Details:
\begin{itemize}
\item Designed and implemented a priority-based preemptive CPU scheduler with randomized tie-breaking for\newline{}equal-priority processes
\item Implemented worst-fit memory allocation strategy with recursive cleanup of zombie processes and their descendants
\item Simulated FCFS (First-Come-First-Serve) disk I/O request queue with realistic process blocking and unblocking
\item Managed process creation (fork), waiting, and cascading termination using a dynamic PCB table and child tracking
\item Used basic containers only (e.g., vectors and arrays); no maps or advanced STL structures
\item Core logic based on textbook chapters 1--10, prioritizing clarity and correctness over performance
\end{itemize}}
